BXjalipsum Package
==================

LaTeX: DUmmy text in Japanese
                                                            
### System requirement

  * TeX format: LaTeX.
  * TeX engine: Anything.
  * Dependent packages: None.

### Installation

  - `*.sty` → $TEXMF/tex/latex/BXjalisum
